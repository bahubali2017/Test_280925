PROJECT GOAL:
Create a clean, standalone MVP for **Anamnesis - MAIA** (Medical AI Assistant). The MVP should include:

1. A secure, minimal login screen (email + password mock auth, local session).
2. A working AI assistant page with a chat interface connected to a DeepSeek API via a JavaScript handler. 
3. JSDoc annotations across all files.
4. ESLint-compliant codebase with zero `eslint-disable` directives.
5. Full reuse of the Tailwind + CSS theme system from the original Anamnesis app to match the look and feel.

TECH REQUIREMENTS:
- Use **Vite + React + Tailwind CSS** for frontend.
- Use **JavaScript or TypeScript** with full JSDoc support.
- Backend (if needed) can be minimal Express or skip completely with mock responses.
- All components should be strictly ESLint clean and mobile-responsive.
